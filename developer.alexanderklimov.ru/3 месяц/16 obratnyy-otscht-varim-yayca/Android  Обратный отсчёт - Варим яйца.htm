Обратный отсчёт - Варим яйца

Третий месяц обучения (закрытая зона)

В системе Android есть абстрактный класс CountDownTimer позволяющий организовать обратный отсчёт.

Для первого знакомства напишем пример отсчёта с 10 секунд до 0.

Смоделируем центр управления космическими полётами. Нам понадобится текстовая метка, в котором будет выводиться число оставшихся секунд. Когда счётчик дойдёт до 0, то появится надпись: Поехали!


package ru.alexanderklimov.helloworld;

import android.os.Bundle;
import android.app.Activity;
import android.view.View;
import android.widget.TextView;
import android.os.CountDownTimer;

public class MainActivity extends Activity {

	TextView tvHello;
	protected CountDownTimer timer;

	@Override
	public void onCreate(Bundle savedInstanceState) {
		super.onCreate(savedInstanceState);
		setContentView(R.layout.activity_main);

		tvHello = (TextView) findViewById(R.id.textView1);
		timer = new MyCounter(10000, 1000);
		timer.start();

	}

	public void butCancel_Click(View v) {
		timer.cancel();
	}

	public class MyCounter extends CountDownTimer {

		public MyCounter(long millisInFuture, long countDownInterval) {
			super(millisInFuture, countDownInterval);
		}

		@Override
		public void onFinish() {
			tvHello.setText("Поехали!");
		}

		@Override
		public void onTick(long millisUntilFinished) {
			tvHello.setText((String.valueOf(millisUntilFinished / 1000)));
		}
	}
}

Мы создали свой класс MyCounter, который происходит от абстрактного класса CountDownTimer, в котором используются два метода. Метод onTick() срабатывает через одинаковый интервал, а метод onFinish() вызывается при окончании отсчёта.

Также я добавил на форму кнопку, позволяющую остановить отсчёт при помощи метода cancel().

Варка яиц

Теперь рассмотрим более продвинутый пример.

Чтобы отследить время варки яиц многие смотрят на часы. Более продвинутые пользуются специальными кухонными таймерами. А мы будем пользоваться телефоном и напишем специальное приложение для этих целей. Естественно, вы можете переделать программу под себя и контролировать, например, время заварки чая и другие глупые вещи, которые не приносят пользы котам.

Создаём новый проект EggTimer и приступим к разработке пользовательского интерфейса. Самое главное - это наличие таймера с отсчётом времени. Также по бокам добавим кнопки "+" и "-", позволяющие уменьшать или увеличивать значение таймера. Когда пользователь нажмет кнопку Старт, отсчёт времени начнется в течение указанного количества минут. Также пользователь может отменить своё решение, нажав на кнопку ещё раз. Для большей функциональности добавим также подсчёт сваренных яиц.

Прослушка событий

Мы уже знакомы с обработкой нажатий на кнопку. Закрепим материал. Итак, для определения нажатия кнопок нам необходимо реализовать их прослушивание. Android предоставляет интерфейс OnClickListener, определяющий методы, которые будут срабатывать при возникновении события. OnClickListener уведомит ваше приложение, когда пользователь нажимает на определенную кнопку. Кроме того, необходимо сообщить каждой кнопке о OnClickListener через метод setOnClickListener:


import android.view.View.OnClickListener;

public class EggTimerActivity extends Activity implements OnClickListener...

eggAddTime.setOnClickListener(this);
eggDecreaseTime.setOnClickListener(this);
startEgg.setOnClickListener(this);

// Обрабатываем нажатия кнопок
public void onClick(View v) {
	if(v == eggAddTime) // кнопка с плюсом
	      setEggTime(eggTime + 1); // увеличиваем счётчик
	    else if(v == eggDecreaseTime) // кнопка с минусом
	      setEggTime(eggTime -1); // уменьшаем счётчик
	    else if(v == startEgg) { // кнопка запуска таймера
	      if(isEggTimer)
	        stopEgg(); // останавливаем таймер
	      else
	        startEgg(); // запускаем таймер
	    }
}

Далее добавим код, который обрабатывает нажатие конкретной кнопки. Нам также понадобятся новые свойства, позволяющие пользователю установить и отслеживать время варки яиц: сколько варок было сделано и запущен ли таймер. 


protected int eggTime = 3;

Обратите внимание, на класс CountDownTimer, позволяющий легко создавать и запускать счётчик, а также получать уведомления через равные промежутки времени в то время, как идет обратный отсчёт.

Методы

Осталось разработать методы для настройки времени варки, запуска и остановки варки, а также счётчика. Мы можем сразу инициализировать некоторые свойства при помощи методов при запуске приложения.

setEggTime() - устанавливает время варки в минутах

setEggCount() - подсчитывает число сваренных яиц

startEgg() - запускает таймер

stopEgg() - останавливает таймер

Обратите внимание, что в методе startEgg() мы определяем слушателей CountDownTimer (onTick и onFinish). onTick будет вызываться каждые 1000 миллисекунд (1 секунду), в то время как вызов onFinish происходит тогда, когда таймер достигнет нуля (смотри исходник ниже).

Всё готово для запуска. Справляемся с нахлынувшим волнением и запускаем проект. Нажимаем на кнопку Старт и время пошло...

Исходный код


package ru.alexanderklimov.eggtimer;

import android.app.Activity;
import android.os.Bundle;
import android.os.CountDownTimer;
import android.widget.Button;
import android.widget.TextView;
import android.view.View;
import android.view.View.OnClickListener;

public class EggTimerActivity extends Activity implements OnClickListener {
	protected Button eggAddTime;
	protected Button eggDecreaseTime;
	protected Button startEgg;
	protected TextView eggCountLabel;
	protected TextView eggTimeLabel;
	
	// Свойства
	protected int eggTime = 3;
	protected CountDownTimer eggCountDownTimer;
	protected int eggCount = 0;
	protected boolean isEggTimer = false;

	/** Called when the activity is first created. */
	@Override
	public void onCreate(Bundle savedInstanceState) {
		super.onCreate(savedInstanceState);
		setContentView(R.layout.main);

		eggAddTime = (Button) findViewById(R.id.egg_time_up);
		eggDecreaseTime = (Button) findViewById(R.id.egg_time_down);
		startEgg = (Button) findViewById(R.id.egg_start);
		eggCountLabel = (TextView) findViewById(R.id.egg_count_label);
		eggTimeLabel = (TextView) findViewById(R.id.egg_time);
		
		eggAddTime.setOnClickListener(this);
	    eggDecreaseTime.setOnClickListener(this);
	    startEgg.setOnClickListener(this);
		
	    // Инициализируем значения счётчиков
		setEggCount(0);
	    setEggTime(3);
	}

	// Обрабатываем нажатия кнопок
	public void onClick(View v) {
		if(v == eggAddTime) // кнопка с плюсом
		      setEggTime(eggTime + 1); // увеличиваем счётчик
		    else if(v == eggDecreaseTime) // кнопка с минусом
		      setEggTime(eggTime -1); // уменьшаем счётчик
		    else if(v == startEgg) { // кнопка запуска таймера
		      if(isEggTimer)
		        stopEgg(); // останавливаем таймер
		      else
		        startEgg(); // запускаем таймер
		    }
	}
	
	/**
	   * Устанавливаем значение в минутах для варки.
	   * Во время запущенного таймера не работает.
	   * @param minutes - число минут.
	   */
	  public void setEggTime(int minutes) {
	    if(isEggTimer)
	      return;
	 
	    eggTime = minutes;
	 
	    if(eggTime < 1)
	      eggTime = 1;
	    // выводим время варки в текстовую метку
	    eggTimeLabel.setText(String.valueOf(eggTime));
	  }
	  
	  /**
	   * Подсчитывает число сваренных яиц и выводит результат на экран.
	   * @param count - число сваренных яиц
	   */
	  public void setEggCount(int count) {
	    eggCount = count;
	    // выводим число в текстовую метку
	    eggCountLabel.setText(String.valueOf(eggCount));
	  }
	  
	  /**
	   * Запускает таймер
	   */
	  public void startEgg() {
	    // Создаем новый экземпляр класса CountDownTimer для отсчета времени варки
	    eggCountDownTimer = new CountDownTimer(eggTime * 60 * 1000, 1000) {
	      @Override
	      public void onTick(long millisUntilFinished) {
	    	  // выводим оставшееся время в текстовой метке
	          eggTimeLabel.setText(String.valueOf(millisUntilFinished / 1000));
	      }
	 
	      @Override
	      public void onFinish() {
	    	  isEggTimer = false;
	          setEggCount(eggCount + 1);
	 
	          eggTimeLabel.setText("Готово!");
	          startEgg.setText("Пуск");
	      }
	    };
	 
	    eggCountDownTimer.start();
	    startEgg.setText("Стоп");
	    isEggTimer = true;
	  }
	  
	  /**
	   * Останавливает таймер
	   */
	  public void stopEgg() {
	    if(eggCountDownTimer != null)
	      eggCountDownTimer.cancel();
	 
	    isEggTimer = false;
	    startEgg.setText("Пуск");
	  }
}

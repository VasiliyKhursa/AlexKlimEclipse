Штрих-коды различных форматов повсюду окружают нас. В магазинах для считывания штрих-кодов используют специальные считывающие устройства. Но телефон с камерой и нашей программой вполне можете заменить дорогое устройства. Мы сами с усами.

Разработать программу считывания штрих-кодов оказалось на удивление просто. Вы сможете считывать как стандартные штрих-коды на упаковках товара, так и QR-коды, которые набирают популярность.

Barcode Scanning SDK

Эта статья писалась больше года назад. Недавно, зайдя на сайт разработчиков, увидел, что теперь SDK стала платной. Поэтому вам придётся самостоятельно поискать старый файл библиотеки в сети, чтобы попробовать пример. К счастью, есть альтернатива. Во второй части статьи я добавил новый материал с использованием библиотеки с открытыми исходниками ZXing

Для работы нам понадобится дополнительная библиотека. Идём на сайт Barcode Scanning SDK, где можно скачать архив под Android и iOS. В архиве можно найти документацию, демонстрационный пример и саму jar-библиотеку.

Вы можете сразу подключить демонстрационный пример и разобраться самостоятельно. Я же расскажу этот процесс по шагам с нуля.

Далее необходимо написать код. Для главного экрана (BarcodeActivity.java) код состоит из двух операций - при нажатии кнопки запускается второй экран, после возврата из второго экрана в текстовом поле записывается результат сканирования.


package ru.alexanderklimov.barcode;

import com.biggu.barcodescanner.client.android.Intents;

import android.app.Activity;
import android.content.Intent;
import android.os.Bundle;

import android.view.View;
import android.view.View.OnClickListener;
import android.widget.Button;
import android.widget.TextView;

public class BarcodeActivity extends Activity {
	
	private static final int SCANNER_REQUEST_CODE = 0;
	
    /** Called when the activity is first created. */
    @Override
    public void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.main);
        
        Button button = (Button)findViewById(R.id.button1);
        button.setOnClickListener(new OnClickListener() {
 
            public void onClick(View v) {
 
            	Intent intent = new Intent(v.getContext(), ScannerActivity.class);
				intent.putExtra(Intents.Preferences.ENABLE_BEEP, true);
				intent.putExtra(Intents.Preferences.ENABLE_VIBRATE, true);

				((Activity)v.getContext()).startActivityForResult(intent, SCANNER_REQUEST_CODE);
            }
        });
     }
    
    @Override
    public void onActivityResult(int requestCode, int resultCode, Intent data) {
 
    	if (resultCode == Activity.RESULT_OK && requestCode == SCANNER_REQUEST_CODE) {

			Bundle extras = data.getExtras();
			String result = extras.getString("SCAN_RESULT");
			TextView textView = (TextView)findViewById(R.id.txt);
			textView.setText(result);
		}
    }
}

Библиотека ZXing

ZXing - это библиотека с открытыми исходниками, позволяющая сканировать и обрабатывать штрих-коды различных типов.

Продвинутые программисты могут внедрить открытый код в свой проект. Но зачастую в этом нет большой необходимости.

Наша задача сводится к запуску намерения и обработки результата. Нам нужно добавить пару классов и начать работу. На многих устройствах уже установлена необходимая программа для сканирования штрих-кодов. Если такой программы у пользователя нет, то наша программа предложит ему скачать её из магазина приложений.

Внутри созданного пакета создайте два новых класса IntentIntegrator и IntentResult.

Скопируйте код из библиотеки ZXing и вставьте в созданные классы. Код для IntentIntegrator и код для IntentResult.

Сканируем

Подключим компоненты


private TextView formatTxt, contentTxt;

@Override
protected void onCreate(Bundle savedInstanceState) {
	super.onCreate(savedInstanceState);
	setContentView(R.layout.activity_main);

	scanBtn = (Button) findViewById(R.id.scan_button);
	formatTxt = (TextView) findViewById(R.id.scan_format);
	contentTxt = (TextView) findViewById(R.id.scan_content);
}

Напишем обработчик для нажатия на кнопку. Создадим в нём экземпляр импортированного класса IntentIntegrator и запустим метод сканирования.


public void onClick(View v) {
	IntentIntegrator scanIntegrator = new IntentIntegrator(this);
	scanIntegrator.initiateScan();
}

Если на устройстве установлен сканер штрих-кодов, то начнётся процесс сканирования. В противном случае пользователю будет предложено скачать необходимую программу.

Также вы можете выбрать тип штрих-кода. По умолчанию метод пытается распознать все поддерживаемые форматы, включая UPC-A, UPC-E, EAN-8, EAN-13, QR Code, RSS-14, RSS Expanded, Data Matrix, Aztec, PDF 417, Codabar, ITF, Codes 39, 93, и 128. У метода есть перегруженная версия, в параметрах которого нужно указать коллекцию нужных форматов.

Результат сканирования будет возвращён нашей активности и мы можем обработать его в методе onActivityResult().


@Override
protected void onActivityResult(int requestCode, int resultCode,
		Intent intent) {
	// TODO Auto-generated method stub
	super.onActivityResult(requestCode, resultCode, intent);
	IntentResult scanningResult = IntentIntegrator.parseActivityResult(
			requestCode, resultCode, intent);

	if (scanningResult != null) {
		// we have a result
		String scanContent = scanningResult.getContents();
		String scanFormat = scanningResult.getFormatName();
		formatTxt.setText("FORMAT: " + scanFormat);
		contentTxt.setText("CONTENT: " + scanContent);

	} else {
		Toast toast = Toast.makeText(getApplicationContext(),
				"No scan data received!", Toast.LENGTH_SHORT);
		toast.show();
	}
}

Загрузите приложение на реальное устройство и попробуйте сканировать какой-нибудь штрих-код. У вас запустится приложение (если установлено) и вернёт две строчки: формат штрих-кода и содержимое. Дальнейшие действия зависят от вас. Например, можно подключиться к базе данных и узнать информацию, которая содержится в штрих-коде.

Если вы уверены, что нужный сканер штрих-кодов установлен на устройстве, то можете вызвать намерение следующим образом:


public void onClick(View v) {
    Intent intent = new Intent("com.google.zxing.client.android.SCAN");
    intent.putExtra("SCAN_MODE", "QR_CODE_MODE");
    startActivityForResult(intent, 0);
}

public void onActivityResult(int requestCode, int resultCode, Intent intent) {
    if (requestCode == 0) {
        if (resultCode == RESULT_OK) {
            String contents = intent.getStringExtra("SCAN_RESULT");
            String format = intent.getStringExtra("SCAN_RESULT_FORMAT");
            // Handle successful scan
            Toast toast = Toast.makeText(this, "Content:" + contents + " Format:" + format , Toast.LENGTH_LONG);
             toast.show();
        } else if (resultCode == RESULT_CANCELED) {
            // Handle cancel
            Toast toast = Toast.makeText(this, "Scan was Cancelled!", Toast.LENGTH_LONG);
            toast.setGravity(Gravity.TOP, 25, 400);
            toast.show();
        }
    }
}

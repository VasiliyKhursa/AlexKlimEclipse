Секретный код

Третий месяц обучения (закрытая зона)

Возможно, вы знаете, что в телефонах есть секретные коды для вызова различных скрытых программ. Изначально секретные коды использовались для чтения служебной информации инженерами, потом сведения о кодах стала известна широкому кругу пользователей. Небольшая подборка кодов есть здесь.

Для вызова секретного приложения используетя формат *#*#(5цифр)#*#* (вначале звёздочка и решётка, в конце наоборот - решётка и звёздочка).

Мы можем создать свой секретный код для собственного приложения. Делается это очень просто.

Создаём обычное стандартное приложение. Далее добавляем новый класс, который будет наследоваться от BroadcastReceiver, например, MySecretCodeReceiver.java:


package ru.alexanderklimov.secretcode;

import android.content.BroadcastReceiver;
import android.content.Context;
import android.content.Intent;

public class MySecretCodeReceiver extends BroadcastReceiver {
	@Override
	public void onReceive(Context context, Intent intent) {
		if(intent.getAction().equals("android.provider.Telephony.SECRET_CODE")) {
			Intent myintent = new Intent(context, MainActivity.class);
			myintent.addFlags(Intent.FLAG_ACTIVITY_NEW_TASK);
			context.startActivity(myintent);
		}
	}
}

Здесь мы сравнимаем намерение со строчкой android.provider.Telephony.SECRET_CODE и если такая строчка обнаружится, то запускается активность.

Далее открываем файл манифеста и добавляем в области application блок:


<receiver android:name=".MySecretCodeReceiver" >
    <intent-filter>
        <action android:name="android.provider.Telephony.SECRET_CODE" />

        <data
            android:host="79532"
            android:scheme="android_secret_code" />
    </intent-filter>
</receiver>

Здесь мы устанавливаем имя android.provider.Telephony.SECRET_CODE, а также прописываем пятизначный секретный код в android:host и схему android:scheme="android_secret_code".

Всё! Запускайте звонилку, набирайте код в указанном формате и у вас запустится ваше приложение!

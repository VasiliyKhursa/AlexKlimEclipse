Здесь показано окно с эмулятора, на реальном устройстве список мелодий гораздо шире.

RingtoneManager

Android содержит класс RingtoneManager, который обеспечивает доступ к мелодиям, уведомлениям и другие типам звуков. Он управляет запросами различных поставщиков содержимого и объединяет результаты в один курсор.

Чтобы ваше приложение могло изменять рингтон, необходимо указать разрешение android.permission.WRITE_SETTINGS в манифесте файла.

Чтобы установить рингтон по умолчанию на телефон с SD-карты или памяти телефона, используйте метод setActualDefaultRingtoneUri():


Uri SetRing = Uri.parse("file:///ringtonelist/res/raw/Ring.mp3");						
RingtoneManager.setActualDefaultRingtoneUri(context, RingtoneManager.TYPE_NOTIFICATION, SetRing);

Воспроизвести мелодию

Ниже показан пример воспроизведения мелодии на телефоне. Желательно проверять на реальном устройстве, а не на эмуляторе.


try {
	Uri alert = RingtoneManager
			.getDefaultUri(RingtoneManager.TYPE_ALARM);
	MediaPlayer mMediaPlayer = new MediaPlayer();
	mMediaPlayer.setDataSource(this, alert);
	final AudioManager audioManager = (AudioManager) getSystemService(Context.AUDIO_SERVICE);
	if (audioManager.getStreamVolume(AudioManager.STREAM_ALARM) != 0) {
		mMediaPlayer.setAudioStreamType(AudioManager.STREAM_ALARM);
		mMediaPlayer.setLooping(true);
		mMediaPlayer.prepare();
		mMediaPlayer.start();
	}
} catch (Exception e) {
}

В данном примере мы получаем путь к звуку, который используется для будильника (TYPE_ALARM), который затем передаётся музыкальному проигрывателю. Существуют еще типы TYPE_NOTIFICATION и TYPE_RINGTONE.

Режим «Без звука»

У телефона есть возможность установить режим Обычный, Без звука, Вибрация. Сделаем это программно через класс AudioManager. Для простоты примера будем переключаться между двумя режимами: Обычный/Без звука. Добавим на экран компонент ToggleButton.


TextView tvStatus;
ToggleButton toggleState;

@Override
public void onCreate(Bundle savedInstanceState) {
	super.onCreate(savedInstanceState);
	setContentView(R.layout.activity_main);

	tvStatus = (TextView) findViewById(R.id.textView1);
	toggleState = (ToggleButton) findViewById(R.id.toggleButton1);

	final AudioManager mode = (AudioManager) this
			.getSystemService(Context.AUDIO_SERVICE);
	
	if(mode.getRingerMode() == mode.RINGER_MODE_NORMAL)
		toggleState.setChecked(true);
	else
		toggleState.setChecked(false);

	toggleState.setOnCheckedChangeListener(new OnCheckedChangeListener() {
		public void onCheckedChanged(CompoundButton buttonView,
            	boolean isChecked) {
            // TODO Auto-generated method stub
            if (isChecked) {
                tvStatus.setText("Обычный режим");
                mode.setRingerMode(AudioManager.RINGER_MODE_NORMAL);
                Toast.makeText(getBaseContext(), "Установлен обычный режим", 
                        Toast.LENGTH_LONG).show();
            } else {
            	tvStatus.setText("Режим без звука");
            	mode.setRingerMode(AudioManager.RINGER_MODE_SILENT);
            	// mode.setRingerMode(AudioManager.RINGER_MODE_VIBRATE); // виброрежим
            	Toast.makeText(getBaseContext(), "Установлен Режим без звука",
                        Toast.LENGTH_LONG).show();
            }
         }
	});
}

Убедиться, что код работает, можно по значку, который появляется в строке состояния телефона.

Управление громкостью

Для управления громкостью можно использовать тот же AudioManager с методами getStreamVolume() и setStreamVolume().

Добавим в экран активности компонент SeekBar и напишем следующий код:


private SeekBar volumeBar;
private AudioManager audiomanager;

public void onCreate(Bundle savedInstanceState) {
	super.onCreate(savedInstanceState);
	setContentView(R.layout.activity_test);

	volumeBar = (SeekBar) findViewById(R.id.seekBar1); // SeekBar

	audiomanager = (AudioManager) this
			.getSystemService(Context.AUDIO_SERVICE);

	// Настраиваем SeekBar
	volumeBar.setMax(audiomanager
			.getStreamMaxVolume(AudioManager.STREAM_RING)); // максимальный уровень громкости
	volumeBar.setKeyProgressIncrement(1);
	volumeBar.setProgress(audiomanager
			.getStreamVolume(AudioManager.STREAM_RING)); // текущее значение

	// Отслеживаем изменение ползунка
	volumeBar.setOnSeekBarChangeListener(new OnSeekBarChangeListener() {
		@Override
		public void onStopTrackingTouch(SeekBar seekBar) {
		}

		@Override
		public void onStartTrackingTouch(SeekBar seekBar) {
		}

		@Override
		public void onProgressChanged(SeekBar seekBar, int progress,
				boolean fromUser) {
			// меняем уровень громкости и выводим всплывающее окно
			// с текущим уровнем и проигрыванием звука
			audiomanager.setStreamVolume(AudioManager.STREAM_RING,
					progress, AudioManager.FLAG_SHOW_UI
							+ AudioManager.FLAG_PLAY_SOUND);
		}
	});
}

@Override
public boolean onKeyDown(int keyCode, KeyEvent event) {
	// Обработка нажатий кнопок регулировки звука на самом аппарате
	if (keyCode == KeyEvent.KEYCODE_VOLUME_DOWN
			|| keyCode == KeyEvent.KEYCODE_VOLUME_UP) {
		// меняем позицию ползунка
		volumeBar.setProgress(audiomanager
				.getStreamVolume(AudioManager.STREAM_RING));
	}
	return super.onKeyDown(keyCode, event);
}

При изменение уровня громкости через SeekBar, вы меняете системные настройки. Также вы можете отслеживать изменение громкости при нажатии аппаратных кнопок громкости.

Играем мелодию в формате RTTTL (Ring Tone Text Transfer Language)

Кроме проигрывания обычных музыкальных файлов, существует специальный формат RTTTL, разработанный компанией Nokia. Подробности о формате можно поискать в Википедии, а здесь рассмотрим только пример.

Создадим новый файл под именем sample.rtttl в папке /res/raw/ и зададим мелодию в RTTTL-формате.


a4:d=4,o=5,b=250:a4;

Для запуска созданной мелодии пишем следующее:


MediaPlayer mediaPlayer = new MediaPlayer();
if (mediaPlayer != null) {
	mediaPlayer.release();
}

mediaPlayer = MediaPlayer.create(MainActivity.this, R.raw.sample);
mediaPlayer.start();

Если наглая рыжая морда решила зарегистрироваться в социальной сети Книга Морд, которая у тупых американцев называется FaceBook, то надо проделать несколько вещей.

Запускаем официальный клиент

У официального клиента FaceBook, установленного на устройстве, имя пакета com.facebook.katana, а имя класса, отвечающего за вход в систему - com.facebook.katana.LoginActivity. Зная эти данные, мы можем запустить официальное приложение из своей программы. Пока опустим различные проверки для упрощения кода:


public void onClick(View v) {
	Intent intent = new Intent("android.intent.category.LAUNCHER");
	String facebookPackageName = "com.facebook.katana";
	String facebookClassName = "com.facebook.katana.LoginActivity";
	intent.setClassName(facebookPackageName, facebookClassName);
	startActivity(intent);
}

Проверяем и устанавливаем

Если у пользователя не установлен клиент FaceBook, то предыдущий код вызовет ошибку. Кроме того, мы можем предложить пользователю скачать нужную программу, если её ещё нет на компьютере. Исправленная версия будет следующей:


public void onClick(View v) {
	
	String facebookPackageName = "com.facebook.katana";
	String facebookClassName = "com.facebook.katana.LoginActivity";
	
	try {
		// Проверка на установленное приложение
		getPackageManager().getApplicationInfo(facebookPackageName, 0);
		Intent intent = new Intent("android.intent.category.LAUNCHER");
		intent.setClassName(facebookPackageName, facebookClassName);
		startActivity(intent);
	} catch (NameNotFoundException e) {
		Toast.makeText(getApplicationContext(),
				"Фейсбук не найден! Установите приложение.", Toast.LENGTH_LONG).show();

		// Запускаем маркет для загрузки приложения
		Uri uri = Uri.parse("market://details?id=" + facebookPackageName);
		Intent intent = new Intent(Intent.ACTION_VIEW, uri);
		startActivity(intent);
	}
}

Запускаем определённую страницу

Кроме основной страницы пользователя, есть ещё дополнительные страницы для просмотра фотографий, встреч, входящий сообщений, списка друзей и т.д. Стоит отметить, что не все из них доступны через приложение. В этом случае приложение предложит перейти на веб-версию сайта.

Добавим на экран элемент Spinner со списком страниц, которые будем запускать через нажатине кнопки:


public class MainActivity extends Activity {

	Spinner spinnerAction;
	String[] facebookUris = { "facebook:/chat", "facebook:/events",
			"facebook:/friends", "facebook:/inbox", "facebook:/info",
			"facebook:/newsfeed", "facebook:/places", "facebook:/requests",
			"facebook:/wall", "facebook:/photos_stream" };

	@Override
	protected void onCreate(Bundle savedInstanceState) {
		super.onCreate(savedInstanceState);
		setContentView(R.layout.activity_main);

		spinnerAction = (Spinner) findViewById(R.id.spinnerAction);
		ArrayAdapter<String> adapter = new ArrayAdapter<String>(this,
				android.R.layout.simple_spinner_item, facebookUris);
		adapter.setDropDownViewResource(android.R.layout.simple_spinner_dropdown_item);
		spinnerAction.setAdapter(adapter);

	}

	public void onClick(View v) {
		String facebookPackageName = "com.facebook.katana";
		String facebookClassName = "com.facebook.katana.LoginActivity";

		try {
			// Проверка на установленное приложение
			getPackageManager().getApplicationInfo(facebookPackageName, 0);

			spinnerAction.getSelectedItem();
			String action = (String) spinnerAction.getSelectedItem();
			// Toast.makeText(getApplicationContext(), action,
			// Toast.LENGTH_LONG).show();
			Intent intent = new Intent(Intent.ACTION_VIEW, Uri.parse(action));
			startActivity(intent);
		} catch (NameNotFoundException e) {
			Toast.makeText(getApplicationContext(),
					"Фейсбук не найден! Установите приложение.",
					Toast.LENGTH_LONG).show();

			// Запускаем маркет для загрузки приложения
			Uri uri = Uri.parse("market://details?id=" + facebookPackageName);
			Intent intent = new Intent(Intent.ACTION_VIEW, uri);
			startActivity(intent);
		}
	}
}

Facebook SDK for Android - официальная документация по FaceBook

На странице http://developers.facebook.com/android/ есть подробная документация об использовании возможностей FaceBook в своём приложении. Давайте попробуем.

Скачиваем с указанной страницы архив размером 12 мегабайт. На данный момент SDK имеет версию 3.0. Распакуйте архив в отдельную папку.

В папке bin лежит готовый apk-файл, который вы можете установить на эмуляторе. На реальном устройстве проще скачать программу с Google Play. Для использования в своих приложениях данный apk не нужен, но отсутствие файла может вызывать различные ошибки в некоторых сценариях использования Facebook.

Напоминаю, что установить можно через командную строку (пример для Windows 7):


adb install %HOMEPATH%acebook-android-sdk-3.0\bin\FBAndroid-2.0.apk

Убедитесь, что приложение установилось на эмуляторе. Файл большой, поэтому запаситесь терпением. Дождитесь успешной установки, на это требуется несколько минут.

Далее необходимо установить сам SDK. Сам проект для работы с SDK находится в папке facebook, а в папке samples находятся несколько готовых примеров для изучения.

Импортируйте библиотеку SDK и примеры через меню File | Import | General | Existing Projects into Workspace:

Выберите корневую папку facebook-android-sdk-3.0. Eclipse отобразить список доступных проектов.

Когда вы импортируете все проекты, то список примеров будет следующим:

  • FacebookSDK - сама библиотека
  • ProfilePictureSample: demonstrates non-logged-in usage of the profile picture control
  • FriendPickerSample: demonstrates usage of the native friend picker to choose amongst the user's friends
  • PlacePickerSample: demonstrates logged-in usage of the place picker control
  • SessionLoginSample: demonstrates detailed login flow using the SDK
  • JustRequestSample: demonstrates logged-in user making single and batched Facebook API requests
  • BooleanOGSample: demonstrates publishing an Open Graph custom action type
  • SwitchUserSample: demonstrates an approach to allow switching between multiple users
  • HelloFacebookSample: an all-round sample that demonstrates profile access, status updates and photo uploading
  • Scrumptious: demonstrates the use of login, requests, pickers, picture uploads, and Open Graph publishing
  • Hackbook: another sample app demonstrating a selection of Graph API calls. Note that this sample is untouched from v2.0 of the SDK, and demonsrates the SDK's backward-compatibility.

Каждый пример должен иметь ссылку на библиотеку. Щёлкните правой кнопкой мыши на имени проекта-примера и выберите в левой панели пункт Android. Убедитесь, что в разделе Library отмечена флажком библиотека FacebookSDK.

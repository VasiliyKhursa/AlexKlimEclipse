Path (англ.) - путь, символ X символизирует колючку. Получается, что XPath - это тернистый путь кота.

Знакомство

Парсер XPath рекомендован к использованию на веб-сайтах консорциумом W3C. Android имеет встроенную поддержку данной библиотеки и если вы являетесь веб-мастером, то данный парсер может стать удобным решением, который сможете использовать и на сайтах и в своих приложениях.

XPath включает в себя более ста функций для работы со строками, числами, датами, XML-узлами и т.д.

Рассмотрим простейшую структуру файла:


<?xml version="1.0" encoding="UTF-8"?>

<bookstore>
  <book>
    <title lang="en">Harry Potter</title>
    <author>J K. Rowling</author>
    <year>2005</year>
    <price>29.99</price>
  </book>
</bookstore>

В данном файле тег <bookstore> является корневым узлом. Тег <author>J K. Rowling</author> является одним из узлов. Также присутствуют атрибуты узла, например, lang="en".

Каждый узел или атрибут имеет родителя. Например, у элементов title, author, year, price родителем является элемент book. В свою очередь каждый элемент может иметь дочерние элементы или не иметь их.

Ну и так далее. Я думаю, вы так знаете без меня.

Теперь немного о синтаксисе XPath. Ниже представлена небольшая таблица выражений для разбора XML-документа.

Разбор локального XML-документа

Для начала попробуем обработать локальный файл. Создадим в папке res/raw новый файл sample.xml следующего содержания:

res/raw/sample.xml


<?xml version="1.0" encoding="UTF-8"?>
<sample>
	<info>
		<title>Используем XPath для обработки XML</title>
		<author>Александр Климов</author>
	<list>
		<cat id="1">
			<name gender="Кот">Мурзик</name>
			<age>8</age>
		</cat>

		<cat id="2">
			<name gender="Кот">Барсик</name>
			<age>7</age>
		</cat>

		<cat id="3">
			<name gender="Кот">Рыжик</name>
			<age>6</age>
		</cat>
		
		<cat id="4">
		    <name gender="Кошка">Мурка</name>
		    <age>5</age>
		</cat>
	</list>
	</info>
</sample>

В этом документе есть узлы cat, name, age, а также атрибуты id и gender, которые вы можете использовать в экспериментах.

Разместим на экране активности элемент ListView, в котором будем выводить список, заполненный определённым образом. Предположим, мы хотим вывести в списке имена всех котов (узел cat):


// Если этот код работает, его написал Александр Климов,
// а если нет, то не знаю, кто его писал.

ArrayList<String> mCats = new ArrayList<String>();

public void onCreate(Bundle savedInstanceState) {
	super.onCreate(savedInstanceState);
	setContentView(R.layout.activity_test);

	ListView lv = (ListView)findViewById(R.id.listView1);

	try {
		parseData();
	} catch (Exception ex) {
		Toast.makeText(this, "Exception: " + ex.getMessage(),
				Toast.LENGTH_LONG).show();
	}

	// адаптер с данными, полученными через XPath для вывода в ListView
	ArrayAdapter<String> adapter = new ArrayAdapter<String>(this,
			android.R.layout.simple_list_item_1, mCats);
	lv.setAdapter(adapter);
}

private void parseData() throws Exception {
	// создаем объект InputSource из файла /res/raw/sample.xml
	InputSource inputSrc = new InputSource(getResources().openRawResource(
			R.raw.sample));
	// Создаем экземпляр XPath
	XPath xpath = XPathFactory.newInstance().newXPath();
	// задаем выражение для разбора
	String expression = "//name";
	// список полученных узлов
	NodeList nodes = (NodeList) xpath.evaluate(expression, inputSrc,
			XPathConstants.NODESET);

	Toast.makeText(this, "всего: " + String.valueOf(nodes.getLength()),
			Toast.LENGTH_SHORT).show();
	// если узел найден
	if (nodes != null && nodes.getLength() > 0) {
		mCats.clear();
		int len = nodes.getLength();
		for (int i = 0; i < len; ++i) {
			// query value
			Node node = nodes.item(i);
			// формируем списочный массив
			mCats.add(node.getTextContent());
		}
	}
}

Чтобы в ListView вывести список имён котов, нужно сформировать списочный массив ArrayList и передать его в адаптер. Для формирования массива служит метод parseData(), в котором происходит разбор документа.

Сначала создаётся объект типа InputSource для приёма данных - это может быть файл из ресурсов, файловый поток, строка. Далее создаётся сам XPath для обработки документа. После чего нужно указать выражение, по которому идёт разбор документа. Мы знаем, что имена котов в документе содержатся в тегах name, поэтому задаём соответствующее выражение:


String expression = "//name";

Если мы хотим вывести в список только возраст котов, то тогда используем другое выражение:


String expression = "//age";

А если хотим видеть сразу имя и возраст, то попробуйте выражение:


String expression = "//cat";

Если хотим вывести атрибут gender, то используем выражение:


String expression = "//@gender";

Разбор XML-документа из интернета

Принципиальной разницы разбора XML-документа, который находится на сайте, нет. Единственно, что надо учитывать - многопоточность. Я взял пример из урока HttpURLConnection: загрузка страницы и немного его переделал. Парсить будем мой RSS-канал, который тоже является XML-документом. Получив файловый поток, мы скормим его XPath.

Я решил добавить в пример кнопку, по нажатию которой и будет происходить соединение с сайтом, получение поток и разбор файла.

Для разбора использовалось выражение "//title", которое содержит заголовки уроков.

Полный код примера будет следующим:


// Если этот код работает, его написал Александр Климов,
// а если нет, то не знаю, кто его писал.
package ru.alexanderklimov.xpath;

import ...

public class MainActivity extends Activity {

	ListView lv;
	ArrayList<String> mCats = new ArrayList<String>();
	// Адрес RSS-канала
	private String bestUrl = "http://feeds.feedburner.com/alexanderklimov/VJcl?format=xml";

	@Override
	protected void onCreate(Bundle savedInstanceState) {
		super.onCreate(savedInstanceState);
		setContentView(R.layout.activity_main);

		lv = (ListView) findViewById(R.id.listView1);
	}

	private class DownloadPageTask extends
			AsyncTask<String, Void, List<String>> {

		@Override
		protected ArrayList<String> doInBackground(String... urls) {
			try {
				return downloadOneUrl(urls[0]);
			} catch (Exception e) {
				e.printStackTrace();
			}
			return mCats;
		}

		@Override
		protected void onPostExecute(List<String> result) {
			lv.setAdapter(new ArrayAdapter<String>(MainActivity.this,
					android.R.layout.simple_list_item_1, result));
			super.onPostExecute(result);
		}
	}

	private ArrayList<String> downloadOneUrl(String myurl) throws IOException,
			Exception {
		InputStream inputstream = null;

		try {
			URL url = new URL(myurl);
			HttpURLConnection connection = (HttpURLConnection) url
					.openConnection();
			connection.setReadTimeout(100000);
			connection.setConnectTimeout(100000);
			connection.setRequestMethod("GET");
			connection.setInstanceFollowRedirects(true);
			connection.setUseCaches(false);
			connection.setDoInput(true);

			int responseCode = connection.getResponseCode();

			if (responseCode == HttpURLConnection.HTTP_OK) { // 200 OK
				inputstream = connection.getInputStream();

				InputSource inputSrc = new InputSource(inputstream);
				// Создаем экземпляр XPath
				XPath xpath = XPathFactory.newInstance().newXPath();
				// задаем выражение для разбора
				String expression = "//title";
				// список полученных узлов
				NodeList nodes = (NodeList) xpath.evaluate(expression,
						inputSrc, XPathConstants.NODESET);

				// если узел найден
				if (nodes != null && nodes.getLength() > 0) {
					mCats.clear();
					int len = nodes.getLength();
					for (int i = 0; i < len; ++i) {
						// формируем списочный массив
						Node node = nodes.item(i);
						mCats.add(node.getTextContent());
					}
				}

			} else {
				String data = connection.getResponseMessage()
						+ " . Error Code : " + responseCode;
			}
			connection.disconnect();
			// return data;
		} catch (MalformedURLException e) {
			// TODO Auto-generated catch block
			e.printStackTrace();
		} catch (IOException e) {
			// TODO Auto-generated catch block
			e.printStackTrace();
		} finally {
			if (inputstream != null) {
				inputstream.close();
			}
		}
		return mCats;
	}

	public void onClick(View view) {
		ConnectivityManager myConnMgr = (ConnectivityManager) getSystemService(Context.CONNECTIVITY_SERVICE);
		NetworkInfo networkinfo = myConnMgr.getActiveNetworkInfo();

		if (networkinfo != null && networkinfo.isConnected()) {
			new DownloadPageTask().execute(bestUrl);
		} else {
			Toast.makeText(this, "Нет интернета", Toast.LENGTH_SHORT).show();
		}
	}

	@Override
	public boolean onCreateOptionsMenu(Menu menu) {
		// Inflate the menu; this adds items to the action bar if it is present.
		getMenuInflater().inflate(R.menu.activity_main, menu);
		return true;
	}
}

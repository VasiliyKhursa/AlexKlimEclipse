В некоторых примерах мы видели, как можно было из своего приложения отправить текст в другое приложение, например, текст в программу для отправки SMS или электронного письма. Рассмотрим теперь обратную задачу - возможность приёма данных из другого приложения.

Мы рассмотрим пример с приёмом текстовых данных и графических изображений - это наиболее распространённые данные. Поняв принцип, вы сможете обрабатывать и другие данные.

Intent-фильтры

Вся магия заключается в специальных фильтрах, которые прописаны в манифесте. Если вы откроете манифест у любой стандартной программы, то там уже есть один фильтр:


<intent-filter>
    <action android:name="android.intent.action.MAIN" />

    <category android:name="android.intent.category.LAUNCHER" />
</intent-filter>

Данный фильтр отвечает за запуск вашего приложения, когда вы нажимаете на значок приложения в программе Приложения.

Вам нужно добавить подходящий фильтр для приёма данных в манифесте, используя тег <intent-filter>. Например, если ваше приложение будет обрабатывать получение текста, одного изображения любого типа или нескольких изображений любого типа, то следует добавить следующие фильтры:


<intent-filter>
    <action android:name="android.intent.action.SEND" />

    <category android:name="android.intent.category.DEFAULT" />

    <data android:mimeType="image/*" />
        </intent-filter>
        <intent-filter>
    <action android:name="android.intent.action.SEND" />

    <category android:name="android.intent.category.DEFAULT" />

    <data android:mimeType="text/plain" />
</intent-filter>

<intent-filter>
    <action android:name="android.intent.action.SEND_MULTIPLE" />

    <category android:name="android.intent.category.DEFAULT" />

    <data android:mimeType="image/*" />
</intent-filter>

Как видите, мы указываем имя action.Send, а также указываем тип обрабатываемых данных в теге <data android:mimeType>.

Теперь, когда другое приложение пытается поделиться каким-либо из этих типов данных, используя метод startActivity(), ваше приложение будет находиться в списке выбора приложений. Если пользователь выберет ваше приложение, то соответствующая активность из вашей программы будет запущена. Вам остаётся только порадоваться за выбор пользователя и обработать поступившиеся данные.

Обработка текстовых данных

Итак, начнём по порядку. Cоздайте новое приложение и добавьте указанные выше фильты в манифест.

На основной экран приложения добавьте текстовое поле для приёма текстовых сообщений.

Чтобы обработать данные, поступающие с помощью Intent от другого приложения, используйте метод getIntent() для получения этого объекта Intent. Получив объект, вы можете проверить его содержимое, чтобы определить, что делать дальше. Для этого используются методы getAction() и getType(). Имейте в виду, что если эта активность может быть запущена из других частей системы, таких как Launcher, то вам необходимо будет принять меры для обработки.


EditText editReceiver;

@Override
protected void onCreate(Bundle savedInstanceState) {
	super.onCreate(savedInstanceState);
	setContentView(R.layout.activity_main);

	editReceiver = (EditText) findViewById(R.id.editReceiver);

	// получаем intent, action и MIME-тип
	Intent intent = getIntent();
	String action = intent.getAction();
	String type = intent.getType();

	if (Intent.ACTION_SEND.equals(action) && type != null) {
		if ("text/plain".equals(type)) {
			handleSendText(intent); // Handle text being sent
		}
	} else {
		// обработка других намерений, например, из экрана Home
	}
}

void handleSendText(Intent intent) {
	String sharedText = intent.getStringExtra(Intent.EXTRA_TEXT);
	if (sharedText != null) {
		// Update UI to reflect text being shared
		editReceiver.setText(sharedText);
	}
}

Напомню, что приложения запускаются через намерение ACTION_MAIN, поэтому можно использовать такой подход:


// если ACTION_SEND
if(receivedAction.equals(Intent.ACTION_SEND)){
    // получен контент от другого приложения
}
else if(receivedAction.equals(Intent.ACTION_MAIN)){
    // приложение запускается обычным способом
}

Обработка графических данных

Закроем браузер и откроем приложение Галерея. Выберем понравившуюся картинку и опять воспользуемся командой Отправить:

У нас снова появится знакомое диалоговое окно, где будет присутствовать значок нашей программы. Выбираем нашу программу, однако ничего не происходит. Что-то случилось? Наше приложение появилось в списке приложений только потому, что мы задали фильтр для обработки изображений (<data android:mimeType="image/*" />). Однако, мы не позаботились об обработке даннных указанного типа. Непорядок. Поэтому перепишем пример, добавив обработку всех созданных ранее фильтров:


package ru.alexanderklimov.getcontent;

import java.util.ArrayList;

import android.net.Uri;
import android.os.Bundle;
import android.app.Activity;
import android.content.Intent;
import android.view.Menu;
import android.widget.EditText;
import android.widget.ImageView;

public class MainActivity extends Activity {

	EditText editReceiver;
	ImageView imgReceiver;

	@Override
	protected void onCreate(Bundle savedInstanceState) {
		super.onCreate(savedInstanceState);
		setContentView(R.layout.activity_main);

		editReceiver = (EditText) findViewById(R.id.editReceiver);
		imgReceiver = (ImageView) findViewById(R.id.imgReceiver);

		// получаем intent, action и MIME-тип
		Intent intent = getIntent();
		String action = intent.getAction();
		String type = intent.getType();

		if (Intent.ACTION_SEND.equals(action) && type != null) {
			if ("text/plain".equals(type)) {
				handleSendText(intent); // Handle text being sent
			} else if (type.startsWith("image/")) {
				handleSendImage(intent); // Handle single image being sent
			}
		} else if (Intent.ACTION_SEND_MULTIPLE.equals(action) && type != null) {
			if (type.startsWith("image/")) {
				handleSendMultipleImages(intent); // Handle multiple images
													// being sent
			}
		} else {
			// обработка других намерений, например, из экрана Home
		}
	}

	void handleSendText(Intent intent) {
		String sharedText = intent.getStringExtra(Intent.EXTRA_TEXT);
		if (sharedText != null) {
			// Update UI to reflect text being shared
			editReceiver.setText(sharedText);
		}
	}

	void handleSendImage(Intent intent) {
		Uri imageUri = (Uri) intent.getParcelableExtra(Intent.EXTRA_STREAM);
		if (imageUri != null) {
			// Update UI to reflect image being shared
			imgReceiver.setImageURI(imageUri);
		}
	}

	void handleSendMultipleImages(Intent intent) {
		ArrayList<Uri> imageUris = intent
				.getParcelableArrayListExtra(Intent.EXTRA_STREAM);
		if (imageUris != null) {
			// Update UI to reflect multiple images being shared
		}
	}

	@Override
	public boolean onCreateOptionsMenu(Menu menu) {
		// Inflate the menu; this adds items to the action bar if it is present.
		getMenuInflater().inflate(R.menu.activity_main, menu);
		return true;
	}

}

Добавим на экран элемент ImageView, который будет принимать изображение, выбранное из другого приложения. Повторяем операции с Галереей и на этот раз наше приложение запустится с выбранной картинкой.

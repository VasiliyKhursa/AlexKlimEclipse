Одним из главных отличий в программировании для настольных и мобильных систем является необходимость учитывать заряд батареи в телефоне. Некоторые приложения очень прожорливы и потребляют слишком много ресурсов. Нужно бережно относиться к расходу батареи, чтобы не огорчать пользователя.

Настройка уровня заряда батареи в эмуляторе

По умолчанию, в эмуляторе показывается 50% заряда батареи. Но вы можете изменить эти показания для тестирования своих приложений, которые должны реагировать на уровень заряда.

Сначала запустите эмулятор. В заголовке эмулятора вы увидите четырёхзначное число, которое обозначает порт сетевого адреса. Зная этот порт, можно подключиться к эмулятору через telnet.

В Windows 7 по умолчанию telnet отключен. Необходимо сначала его включить в настройках.

Запускаем командную строку cmd.exe и вводим команду:

telnet localhost 5554

В случае успешного соединения вы получите сообщение OK и можете вводить команды для эмулятора. Например, чтобы получить справку по командам, которые относятся к батарее, нужно ввести команду:

help power

Нас интересует команда для изменения заряда power capacity. Вы можете установить необходимый уровень заряда в батарее, причем изменения сразу вступают в силу, что сразу видно на значке батареи в эмуляторе

power capacity 100


power capacity 20

Получаем показания батареи в реальном времени

Теперь, когда мы научились устанавливать уровень заряда, можем написать программу, которая в реальном времени будет отслеживать показания заряда и другие параметры, которые можно вывести в текстовой метке.

В Android есть специальный механизм для опрашивания различных событий - Broadcast. C его помощью мы будем всегда в курсе о текущем состоянии батареи. Создадим новый стандартный проект.

И напишем код:


package ru.alexanderklimov.battery;

import android.os.BatteryManager;
import android.os.Bundle;
import android.app.Activity;
import android.content.BroadcastReceiver;
import android.content.Context;
import android.content.Intent;
import android.content.IntentFilter;
import android.widget.TextView;

public class MainActivity extends Activity {

	private TextView batteryLevel, batteryVoltage, batteryTemperature,
			batteryTechnology, batteryStatus, batteryHealth, batteryCurrent,
			batteryCharge;

	/** Called when the activity is first created. */
	@Override
	public void onCreate(Bundle savedInstanceState) {
		super.onCreate(savedInstanceState);
		setContentView(R.layout.activity_main);

		batteryLevel = (TextView) findViewById(R.id.level);
		batteryVoltage = (TextView) findViewById(R.id.voltage);
		batteryTemperature = (TextView) findViewById(R.id.temperature);
		batteryTechnology = (TextView) findViewById(R.id.techology);
		batteryStatus = (TextView) findViewById(R.id.status);
		batteryHealth = (TextView) findViewById(R.id.health);
		batteryCurrent = (TextView)findViewById(R.id.current);
		batteryCharge = (TextView) findViewById(R.id.charge);

		this.registerReceiver(this.myBatteryReceiver, new IntentFilter(
				Intent.ACTION_BATTERY_CHANGED));
	}

	private BroadcastReceiver myBatteryReceiver = new BroadcastReceiver() {

		@Override
		public void onReceive(Context context, Intent intent) {
			// TODO Auto-generated method stub
			if (intent.getAction().equals(Intent.ACTION_BATTERY_CHANGED)) {
				batteryLevel.setText("Уровень заряда: "
						+ String.valueOf(intent.getIntExtra("level", 0)) + "%");
				batteryVoltage.setText("Вольтаж: "
						+ String.valueOf((float) intent.getIntExtra("voltage",
								0) / 1000) + "V");
				batteryTemperature.setText("Температура батареи: "
						+ String.valueOf((float) intent.getIntExtra(
								"temperature", 0) / 10) + "°C");
				batteryTechnology.setText("Тип батареи: "
						+ intent.getStringExtra("technology"));
				
				// альтернативный способ вычисления текущего уровня зарядки
				int level = intent.getIntExtra(BatteryManager.EXTRA_LEVEL, -1);
				int scale = intent.getIntExtra(BatteryManager.EXTRA_SCALE, -1);

				float batteryPct = level / (float)scale;
				batteryCurrent.setText("Уровень зарядки: " + batteryPct);

				int status = intent.getIntExtra("status",
						BatteryManager.BATTERY_STATUS_UNKNOWN);
				String strStatus;
				if (status == BatteryManager.BATTERY_STATUS_CHARGING) {
					strStatus = "Заряжается";
				} else if (status == BatteryManager.BATTERY_STATUS_DISCHARGING) {
					strStatus = "Разряжается";
				} else if (status == BatteryManager.BATTERY_STATUS_NOT_CHARGING) {
					strStatus = "Не заряжается";
				} else if (status == BatteryManager.BATTERY_STATUS_FULL) {
					strStatus = "Полный заряд";
				} else {
					strStatus = "Ничего не понимаю";
				}
				batteryStatus.setText("Статус: " + strStatus);

				int health = intent.getIntExtra("health",
						BatteryManager.BATTERY_HEALTH_UNKNOWN);
				String strHealth;
				if (health == BatteryManager.BATTERY_HEALTH_GOOD) {
					strHealth = "Хорошее";
				} else if (health == BatteryManager.BATTERY_HEALTH_OVERHEAT) {
					strHealth = "Перегретое";
				} else if (health == BatteryManager.BATTERY_HEALTH_DEAD) {
					strHealth = "Убитое";
				} else if (health == BatteryManager.BATTERY_HEALTH_OVER_VOLTAGE) {
					strHealth = "Over Voltage";
				} else if (health == BatteryManager.BATTERY_HEALTH_UNSPECIFIED_FAILURE) {
					strHealth = "Unspecified Failure";
				} else {
					strHealth = "Неизвестно";
				}
				batteryHealth.setText("Состояние батареи: " + strHealth);
				
				// Каким образом проходит зарядка?
				int chargePlug = intent.getIntExtra(
						BatteryManager.EXTRA_PLUGGED,
						BatteryManager.BATTERY_STATUS_UNKNOWN);
				String charge = "";
				if (chargePlug == BatteryManager.BATTERY_PLUGGED_USB) {
					charge = "Заряжается от USB";
				} else if (chargePlug == BatteryManager.BATTERY_PLUGGED_AC) {
					charge = "Заряжается от сети";
				} else if (chargePlug == BatteryManager.BATTERY_PLUGGED_WIRELESS) {
					charge = "Беспроводная зарядка";
				} else {
					charge = "Неизвестно";
				}
				batteryCharge.setText("Тип зарядки: " + charge);
			}
		}
	};
}

Запустите проект и посмотрите на результаты

Обратите внимание на первую и последнюю строчку на экране приложения. В первой строке выводится текущий заряд в процентах, в последней - в виде дробной части. Последний вариант я показал на всякий случай, хотя смысла в нём не вижу. Также обратите внимание, что желательно использовать готовые константы. Например, строке level соответствует константа BatteryManager.EXTRA_LEVEL. Поправьте код самостоятельно.

Изменяйте через telnet значения заряда батареи, и вы увидите, что в текстовой метке также будет изменяться значение заряда.

Если устройство подключено к сети переменного тока, то фоновые обновления можно выполнять с максимальной частотой. Если устройство заряжается через USB, частоту можно несколько сократить, а если устройство не подключено к сети – сократить еще больше.

Отслеживание низкого уровня заряда батареи

Постоянно отслеживать показания батареи не самая лучшая идея, поскольку происходит дополнительная нагрузка на батарею. Рекомендуется отслеживать только существенные изменения уровня заряда, в частности, переход устройства в состояние низкого заряда и обратно.

Перепишем предыдущий пример. Создадим новый класс, наследующий от BroadcastReceiver:


package ru.alexanderklimov.battery;

import android.content.BroadcastReceiver;
import android.content.Context;
import android.content.Intent;
import android.widget.Toast;

public class BatteryLevelReceiver extends BroadcastReceiver {

	@Override
	public void onReceive(Context context, Intent intent) {
		// TODO Auto-generated method stub
		boolean batteryLow = intent.getAction().equals(
				Intent.ACTION_BATTERY_LOW);

		if (batteryLow)
			Toast.makeText(
					context,
					"Слишком низкий заряд. Устройство тоже нужно кормить, как и кота",
					Toast.LENGTH_LONG).show();
	}

}

Далее в манифесте зарегистрируем нужные сообщения, которые требуется отслеживать (в разделе application):


<receiver android:name=".BatteryLevelReceiver" >
    <intent-filter>
        <action android:name="android.intent.action.ACTION_BATTERY_LOW" />
        <action android:name="android.intent.action.ACTION_BATTERY_OKAY" />
    </intent-filter>
</receiver>

Для примера я использую только момент, когда заряд становится слишком низким (ACTION_BATTERY_LOW), в этом случае нужно прекратить или уменьшить какие-то сложные операции в приложении. В нашей программе просто выводится всплывающее сообщение Слишком низкий заряд. Устройство тоже нужно кормить, как и кота.

Перейдём в класс активности, удалим код класса myBatteryReceiver и напишем новый код:


package ru.alexanderklimov.battery;

import ...

public class MainActivity extends Activity {

	private TextView batteryLevel, batteryVoltage, batteryTemperature,
			batteryTechnology, batteryStatus, batteryHealth, batteryCurrent;

	/** Called when the activity is first created. */
	@Override
	public void onCreate(Bundle savedInstanceState) {
		super.onCreate(savedInstanceState);
		setContentView(R.layout.activity_main);

		BroadcastReceiver batteryLevelReceiver = new BatteryLevelReceiver();
		IntentFilter filter = new IntentFilter(Intent.ACTION_BATTERY_LOW);
		this.registerReceiver(batteryLevelReceiver, filter);
	}
}

Чтобы увидеть сообщение на эмуляторе, мне пришлось в telnet сначала отключить питание от сети командой:

power ac off

А затем установить уровень заряда на 10 процентов (или ниже):

power capacity 10

Только учтите, что сама система тоже отслеживает заряд батареи таким же образом. Поэтому на эмуляторе системное диалоговое окно частично перекрыло моё всплывающее сообщение (но конец фразы увидеть можно).

По аналогии можете отслеживать момент, когда заряд батареи станет нормальным, чтобы снова включить какие-то ресурсоёмкие операции.

Отслеживание подсоединения к электросети

Аналогично можно отслеживать момент подключения/отключения питания от электросети. Напишем новый класс или добавим в предыдущий пример:


package ru.alexanderklimov.battery;

import android.content.BroadcastReceiver;
import android.content.Context;
import android.content.Intent;
import android.widget.Toast;

public class PowerConnectionReceiver extends BroadcastReceiver {

	@Override
	public void onReceive(Context context, Intent intent) {
		// TODO Auto-generated method stub

		if (intent.getAction().equals(Intent.ACTION_POWER_CONNECTED)) {
			Toast.makeText(context, "Устройство заряжается", Toast.LENGTH_LONG)
					.show();
		} else if (intent.getAction().equals(Intent.ACTION_POWER_DISCONNECTED)) {
			Toast.makeText(context, "Устройство не заряжается", Toast.LENGTH_LONG)
			.show();
		}
	}
}

В манифест добавим нужные фильтры:


<receiver android:name=".PowerConnectionReceiver" >
    <intent-filter>
        <action android:name="android.intent.action.ACTION_POWER_CONNECTED" />
        <action android:name="android.intent.action.ACTION_POWER_DISCONNECTED" />
    </intent-filter>
</receiver>

package ru.alexanderklimov.batterywidget;

import android.appwidget.AppWidgetManager;
import android.appwidget.AppWidgetProvider;
import android.content.ComponentName;
import android.content.Context;
import android.content.Intent;
import android.widget.RemoteViews;

public class BatteryWidgetProvider extends AppWidgetProvider {
	@Override
	public void onUpdate(Context context, AppWidgetManager appWidgetManager,
			int[] appWidgetIds) {
		// TODO Auto-generated method stub
		updateWidget(context);
		context.startService(new Intent(context, MyBatteryReceiver.class));
	}

	public void updateWidget(Context context) {
		RemoteViews updateViews = new RemoteViews(context.getPackageName(),
				R.layout.batterywidget_layout);
		updateViews.setTextViewText(R.id.level, "waiting!");

		ComponentName myComponentName = new ComponentName(context,
				BatteryWidgetProvider.class);
		AppWidgetManager manager = AppWidgetManager.getInstance(context);
		manager.updateAppWidget(myComponentName, updateViews);
	}
}

В созданном классе мы запускаем службу MyBatteryReceiver. Напишем новый класс для службы:


package ru.alexanderklimov.batterywidget;

import android.app.Service;
import android.appwidget.AppWidgetManager;
import android.content.BroadcastReceiver;
import android.content.ComponentName;
import android.content.Context;
import android.content.Intent;
import android.content.IntentFilter;
import android.os.BatteryManager;
import android.os.IBinder;
import android.widget.RemoteViews;

public class MyBatteryReceiver extends Service {

	private int batterylevel = 0;
	private String batteryStatus = "";

	private BroadcastReceiver myReceiver = new BroadcastReceiver() {
		@Override
		public void onReceive(Context context, Intent intent) {
			String action = intent.getAction();
			if (action.equals(Intent.ACTION_BATTERY_CHANGED)) {
				batterylevel = intent.getIntExtra("level", 0);

				int status = intent.getIntExtra("status",
						BatteryManager.BATTERY_STATUS_UNKNOWN);
				
				if (status == BatteryManager.BATTERY_STATUS_CHARGING) {
					batteryStatus = "Charging";
				} else if (status == BatteryManager.BATTERY_STATUS_DISCHARGING) {
					batteryStatus = "Discharging";
				} else if (status == BatteryManager.BATTERY_STATUS_NOT_CHARGING) {
					batteryStatus = "Not charging";
				} else if (status == BatteryManager.BATTERY_STATUS_FULL) {
					batteryStatus = "Full";
				} else {
					batteryStatus = "";
				}

				updateAppWidget(context);
			}
		}

		public void updateAppWidget(Context context) {
			RemoteViews updateViews = new RemoteViews(context.getPackageName(),
					R.layout.batterywidget_layout);
			updateViews.setTextViewText(R.id.level, "Bat. Status:
"
					+ "Level: " + batterylevel + "%
" + "Status: "
					+ batteryStatus);

			ComponentName myComponentName = new ComponentName(context,
					BatteryWidgetProvider.class);
			AppWidgetManager manager = AppWidgetManager.getInstance(context);
			manager.updateAppWidget(myComponentName, updateViews);
		}
	};

	@Override
	public IBinder onBind(Intent arg0) {
		// TODO Auto-generated method stub
		return null;
	}

	@Override
	public void onCreate() {
		// TODO Auto-generated method stub
		super.onCreate();

		IntentFilter intentFilter = new IntentFilter();
		intentFilter.addAction(Intent.ACTION_BATTERY_CHANGED);
		registerReceiver(myReceiver, intentFilter);
	}

	@Override
	public void onDestroy() {
		// TODO Auto-generated method stub
		super.onDestroy();
		unregisterReceiver(myReceiver);
	}
}

В методе onCreate() регистрируем широковещательное сообщение с намерением Intent.ACTION_BATTERY_CHANGED. И обновляем виджет в методе myReceiver.onReceive().

Пропишем службу и приёмник широковещательных сообщений в манифесте для секции application:


<receiver android:name=".BatteryWidgetProvider" >

        <action android:name="android.appwidget.action.APPWIDGET_UPDATE" />

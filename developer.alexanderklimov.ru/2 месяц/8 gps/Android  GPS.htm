Проверка доступности GPS

В настройках системы есть пункт, позволяющий включить или отключить доступность GPS. Для проверки доступности настроек используйте следующий код:


public void onCreate(Bundle savedInstanceState) {
	super.onCreate(savedInstanceState);
	setContentView(R.layout.main);
	tvInfo = (TextView) findViewById(R.id.textView1);
	checkEnableGPS();
}

private void checkEnableGPS() {
	String provider = Settings.Secure.getString(getContentResolver(),
			Settings.Secure.LOCATION_PROVIDERS_ALLOWED);
	if (!provider.equals("")) {
		// GPS Enabled
		tvInfo.setText("GPS доступен: " + provider);
	}
		else{
			tvInfo.setText("GPS выключен");
		}
}

Открыть окно «Моё местоположение»

Сама настройка, позволяющая использовать GPS для нахождения местоположения, находится в Настройки | Моё местоположение. Чтобы программно открыть это окно, используйте следующий код (по нажатию кнопки):


public void OnButtonClick(View v){
	Intent gpsOptionsIntent = new Intent(  
            android.provider.Settings.ACTION_LOCATION_SOURCE_SETTINGS);  
    startActivity(gpsOptionsIntent);  
}

Программное включение/выключение GPS

Начиная с Android 3.0 (API 11), появилась возможность программного включения/выключения GPS через намерение.


final Intent intent = new Intent("android.location.GPS_ENABLED_CHANGE");
// Для включения используем true, для выключения false
intent.putExtra("enabled", true);
getApplicationContext().sendBroadcast(intent);

При тестировании на эмуляторе в области уведомлений появился мигающий значок поиска GPS.
